DESCRIPTION 

 FFT - DISCRETE FOURIER TRANSFORM AND INVERSE

 Computes the Discrete Fourier Transform of the input data vector
 using the FFT routine in the TFSA package. The input vector is
 zero-padded if the FFT length is more than the input data
 length. Similarly, the Inverse FFT can be computed by putting the
 Inverse check box on.
